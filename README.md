# GrooveboxIt

Application desktop Python permettant de jouer des sons via le pavé numérique, de séquencer des patterns et des songs.

Inspirée des grooveboxes hardware, elle offre un mode Drum (lecture one-shot de samples WAV), un mode Synthé (pitchs configurables sur un même sample) et un mode Séquenceur (patterns et songs avec tempo BPM, pistes polymétriques et persistance JSON).
